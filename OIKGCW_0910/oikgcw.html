Git parancsok:

  • git init: Ez a parancs inicializálja a Git rendszert a jelenlegi könyvtárban, létrehozva egy új git repository-t.
  • git clone [url]: Ezzel a paranccsal klónozhatod egy létező git repo-t a saját gépedre.
  • git add .: Ez a parancs a megadott fájlt a Git "stage"-re helyezi, ami azt jelenti, hogy felkészíti a fájlt a commit-ra.
  • git status: Ez a parancs információkat ad a jelenlegi állapotról, például mely fájlok vannak a "stage"-en, melyek lettek módosítva, de még nem kerültek a "stage"-re, és melyeket követtünk még nem a Git-ben.
  • git commit -m "első commit": Ez a parancs "commit"-olja az összes változást, amit a git add parancs segítségével "stage"-re helyeztél. A commit üzenet lehetővé teszi, hogy jegyzetet adj hozzá a commit-hoz, ami leírja, hogy milyen változásokat végeztél.
